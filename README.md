# DXServer

## A Scalable Multi-Client Volume Visualization Server

### [Intro](https://docs.google.com/presentation/d/1qXWaEy7tCw9A6jymXJcniUHjlqITfiwI11qqwl1lh3A/edit?usp=sharing)

### Compile
```  
mkdir build/  
cd build/   
cmake ..  
make  
```

### Run
```
cd nodejs/  
node main.js
```

### Benchmark
```
cd nodejs/  
node simulate.js  
```
